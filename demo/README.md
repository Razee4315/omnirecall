# Demo Assets

Place your demo files here:

## Required Files

- **demo.mp4** - Main demo video showing the app in action
  - Recommended: 720p or 1080p
  - Duration: 30-60 seconds
  - Show: Opening with Alt+Space, chatting, adding documents, code output

## Optional Screenshots

- **spotlight.png** - Screenshot of Spotlight mode
- **dashboard.png** - Screenshot of Dashboard mode
- **settings.png** - Screenshot of Settings panel

## Recording Tips

1. Use a clean desktop background
2. Show the cursor position feature (window appearing at cursor)
3. Demonstrate:
   - Opening with Alt+Space
   - Typing a question
   - Getting formatted response (code block, bold text)
   - Adding a document
   - Switching to Dashboard mode
   - Chat history persistence

## Recommended Tools

- [OBS Studio](https://obsproject.com/) - Free screen recording
- [ScreenToGif](https://www.screentogif.com/) - For GIF creation
- [ShareX](https://getsharex.com/) - Screenshots and recordings
